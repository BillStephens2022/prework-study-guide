Prework Study Guide

HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • Section elements break the body into specific sections for easier navigation
  • Header and Footer elements may be added within a body element
  • Unordered lists (ul) are lists using bullet points.
  • Ordered lists (ol) are lists in numerical order
  • Each list item must be within a 'li' tag (regardless if for ordered or unordered lists

CSS

  • CSS adds styles to a website

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: adds changed files in our working directory to staging areas of Git
  • git commit -m "enter notes here": after staging, this commits the changes to the repository along with a message
  • git pull origin main: pulls base branch into your feature branch. If branches already in sync, you will get a msg saying so
  • git push origin feature/add-html: pushes up all features
